add_protocol(self, name, theclass)

 

Registers a new protocol in addition to the built-in 'ssh', 'telnet', and 'dummy' adapters. The given adapter must implement the same interface as protocols.Transport. Note that you need to pass a class, not an instance.

Parameters:
  • name (str) - The protocol name
  • theclass (protocols.Transport) - The class that handles the protocol.

run(self, hosts, function)

 

Add the given function to a queue, and call it once for each host according to the threading options. Use decorators.bind_args() if you also want to pass additional arguments to the callback function.

Returns an object that represents the queued task, and that may be passed to is_completed() to check the status.

Parameters:

shutdown(self, force=0)

 

Stop executing any further jobs. If the force argument is True, the function does not wait until any queued jobs are completed but stops immediately.

After emptying the queue it is restarted, so you may still call run() after using this method.

Parameters:
  • force (bool) - Whether to wait until all jobs were processed.
