get_account_from_etree(node)

 

Given a <account> node, this function returns a Exscript.Account instance. The following XML syntax is expected, whereby the children of <account> are all optional:

   <account name="myaccount">
     <password type="base64">Zm9v</password>
     <authorization-password type="cleartext">bar</authorization-password>
   </account>

The <password> and <authorization-password> tags have an optional type attribute defaulting to 'cleartext'. Allowed values are 'cleartext' and 'base64'.

Parameters:
  • node (lxml.etree.ElementNode) - A <account> element.
Returns: Exscript.Account
The resulting account.

get_accounts_from_etree(node)

 

Given an lxml.etree node, this function looks for <account> tags and returns a list of Exscript.Account instances. The following XML syntax is expected:

   <root>
      <account name="one"/>
      <account name="two">
        ...
      </account>
      ...
   </root>

The individual accounts are parsed using get_account_from_etree().

Parameters:
  • node (lxml.etree.ElementNode) - A node containing <account> elements.
Returns: list(Exscript.Account)
A list of accounts.

       <list-item>bar</list-item>
     </list>
   </argument-list>
Parameters:
  • node (lxml.etree.ElementNode) - A node containing variable elements.
Returns: dict
A map of variables.

get_host_from_etree(node)

 

Given a <host> node, this function returns a Exscript.Host instance. The following XML syntax is expected, whereby the <argument-list> element is optional:

   <host name="otherhost" address="10.0.0.2">
     <argument-list>
       <list name="mylist">
         <list-item>foo</list-item>
         <list-item>bar</list-item>
       </list>
     </argument-list>
   </host>

The arguments are parsed using get_dict_from_etree() and attached to the host using Exscript.Host.set_all().

Parameters:
  • node (lxml.etree.ElementNode) - A <host> element.
Returns: Exscript.Host
The resulting host.

get_hosts_from_etree(node)

 

Given an lxml.etree node, this function looks for <host> tags and returns a list of Exscript.Host instances. The following XML syntax is expected, whereby the <argument-list> element is optional:

   <root>
      <host name="localhost" address="10.0.0.1"/>
      <host name="otherhost" address="10.0.0.2">
        <argument-list>
          <list name="mylist">
            <list-item>foo</list-item>
            <list-item>bar</list-item>
          </list>
        </argument-list>
      </host>
   </root>

The arguments are parsed using get_arguments_from_etree() and attached to the host using Exscript.Host.set().

Parameters:
  • node (lxml.etree.ElementNode) - A node containing <host> elements.
Returns: list(Exscript.Host)
A list of hosts.
