Function Details

to_list(item)

 

If the given item is iterable, this function returns the given item. If the item is not iterable, this function returns a list with only the item in it.

Parameters:
  • item (object) - Any object.
Returns: list
A list with the item in it.

to_host(host)

 

Given a string or a Host object, this function returns a Host object.

Parameters:
  • host (string|Host) - A hostname (may be URL formatted) or a Host object.
Returns: Host
The Host object.

to_hosts(hosts)

 

Given a string or a Host object, or a list of strings or Host objects, this function returns a list of Host objects.

Parameters:
  • hosts (string|Host|list(string)|list(Host)) - One or more hosts or hostnames.
Returns: list[Host]
A list of Host objects.

to_log(log)

 

Given a string, this function returns a new Logfile object. Given any other Log object, this function just returns the same object.

Parameters:
  • log (string|Log) - A logfile name or a Log object.
Returns: Log
The Log object.

to_logs(logs)

 

Given a string or a Log object, or a list of strings or Log objects, this function returns a list of Log objects.

Parameters:
  • logs (string|Log|list(string)|list(Log)) - One or more logs or logfile names.
Returns: list[Log]
A list of Log objects.
