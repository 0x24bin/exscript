Submodules
  • Exscript.util.cast: Handy shortcuts for converting types.

  • Exscript.util.match: Shorthands for regular expression matching.
  • Exscript.util.report: Formatting logs into human readable reports.

  • Exscript.util.start: Quickstart methods for the Exscript queue.
  • Exscript.util.syslog

  • Exscript.util.template: Executing Exscript templates on a connection.
  • Exscript.util.url: Working with URLs (as used in URL formatted hostnames).
