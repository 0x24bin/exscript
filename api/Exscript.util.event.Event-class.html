Class Event

object --+
         |
        Event

A simple signal/event mechanism, to be used like this:

   def mycallback(arg, **kwargs):
       print arg, kwargs['foo']

   myevent = Event()
   myevent.connect(mycallback)
   myevent.emit('test', foo = 'bar')
   # Or just: myevent('test', foo = 'bar')
